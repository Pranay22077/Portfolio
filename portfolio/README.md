## ğŸ¯ Customization

### Personal Information
Update the following files with your information:
- `src/components/Hero.tsx` - Name and introduction
- `src/components/About.tsx` - Personal details and journey
- `src/components/Skills.tsx` - Technical skills and proficiency
- `src/components/Projects.tsx` - Project portfolio
- `src/components/Contact.tsx` - Contact information and social links

### Styling
- **Colors**: Modify CSS variables in `src/app/globals.css`
- **Animations**: Customize in `tailwind.config.ts`
- **Laser Effects**: Adjust laser beam styles in `globals.css`

### Content
- **Projects**: Update the `projects` array in `Projects.tsx`
- **Skills**: Modify `skillCategories` in `Skills.tsx`
- **Social Links**: Update `socialLinks` in `Contact.tsx`
